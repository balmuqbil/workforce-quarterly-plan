⚙ إعداد الربط مع Google Sheets

1 أنشئ جدول Google Sheets جديد

افتح Google Sheets وأنشئ جدول بيانات جديد. أضف هذه العناوين في الصف الأول:

الاسم | الفريق | الشهر | الربع | المنصة/المنتج | نوع المهمة | نسبة الإنشغال | وصف المهمة | الخدمة إن وجدت | تاريخ الإرسال
2 افتح محرر Apps Script

من القائمة العلوية اختر: Extensions > Apps Script

3 الصق الكود التالي

امسح المحتوى الافتراضي والصق هذا الكود:

function doPost(e) { var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet(); var data = JSON.parse(e.postData.contents); var rows = data.rows; var timestamp = new Date().toLocaleString('ar-SA'); for (var i = 0; i < rows.length; i++) { var r = rows[i]; sheet.appendRow([ r.name, r.team, r.month, r.quarter, r.platform, r.taskType, r.occupancy, r.description, r.service, timestamp ]); } return ContentService .createTextOutput(JSON.stringify({ status: "success", count: rows.length })) .setMimeType(ContentService.MimeType.JSON); } function doGet(e) { return ContentService .createTextOutput(JSON.stringify({ status: "ok", message: "Workforce form endpoint is active" })) .setMimeType(ContentService.MimeType.JSON); }
4 انشر كتطبيق ويي

اضغط Deploy > New deployment ثم اختر Web app.
في "Execute as" اختر Me، وفي "Who has access" اختر Anyone.
اضغط Deploy وانسخ الرابط الذي يظهر.
